 Business Problem

Eco Ride is a mobility company that manages customers, scooter models, and transactions.
Management needs insights into customer behavior, product performance, and revenue trends.
SQL JOINs and Window Functions are used to support data-driven decision making.

 Success Criteria

1. Identify top-performing scooter models per region using RANK()
2. Calculate running totals of customer spending using SUM() OVER
3. Analyze month-over-month revenue using LAG() and LEAD()
4. Segment customers into quartiles using NTILE(4)
5. Measure cumulative customer contribution using CUME_DIST()
